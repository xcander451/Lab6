# Анализ и прогнозирование временного ряда: потребление возобновляемой энергии в США

# Лабораторная работа №6
Моделирование временных рядов с помощью ARIMA, SARIMA и SARIMAX
📌 Описание проекта

В рамках работы выполнен анализ временного ряда ежемесячного потребления возобновляемой энергии в США за период с 1973 по 2025 год (всего более 600 наблюдений). Исходные данные содержат информацию по разным секторам экономики и типам возобновляемой энергии (биомасса, гидроэнергия, геотермальная, солнечная, ветровая).

Основные задачи:

    Исследование структуры ряда (тренд, сезонность, стационарность).

    Построение базовой модели ARIMA с ручным подбором параметров.

    Диагностика остатков и улучшение модели.

    Внедрение сезонной компоненты (SARIMA).

    Добавление внешних факторов (SARIMAX): потребление энергии ветра + фиктивные переменные месяца.

    Автоматический подбор параметров с помощью pmdarima и сравнение с ручной моделью.
